API

MLGetAttackerInfo()
Return the attacker machine infomation in a key-value pair. Infomation includes: name, ip, port, location, CPU manufacturer, age, race, sex and marriagee status.
MLSetAutoAttackBack(MLBool *boolean)
Turn on/off auto attack back.
MLForceAttack(MLMachine *innocent)
Fork attack the machine specified by innoccent paramter.
MLIsWorkingProperly()
Return true, always.
MLUpdate()
Do nothing. majestic llama determines when to update itself based on its mood.
MLSelfDestruct()
Do nothing. You really think you can control majestic llama
MLBash()
Command line mode. Require authorization from both Llama Powder and Zappos.
MLDefendMode()
Hidden command. Triggers guard behavior which form a Llama Power field around the machine. Numerous research results show that the guard behavior is effective in defending aliens, creators from future and creators with super power, espcially zombies.
INVERT_MODE_SECRET_CODE_THE_BEAST()
Hidden command. DO NOT USE IF NOT EMERGENCY. Unlock majestic llama's resource limit. Decompose both hardware and software of the attacker machine into subatom. May cause black hole.
