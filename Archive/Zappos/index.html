It's all about majestic llama

About

majestic llama is a XSS firewall API powered by Llama Power. While originated in North America about 40 million years ago (version 0.0.1), majestic llama change its developping environment to South America about 3 million years ago (version 1.0.0) due to better network coverage.

majestic llama has been downloaded for over 158,000 times in United States and Canada

Stable Version: 5.1.2

Beta Version: 5.2.1

Trusted by Zappos

Features

  • Single Thread

    majestic llama supports only one thread, which makes it very reliable. Besides the security, single thread running environment makes majestic llama extremely fast.

  • 100% Coverage

    majestic llama supports all platforms and languages. You can use API freely, wherever you want, whenever you want. Some highlighted platforms: OS X, MacOS, Linux, BSD, Windows 10, OS/360, DOS, iOS, Android, Windows Mobile 5, Meego, Bada, BlackBerry, webOS, tvOS. Some highlighted languages: ECMAScript, C, Objective-C, C++, Smalltalk, Java, PHP, Python, Fortran, vim script, awk.

  • Anti-attack Mechanism

    • With its built-in technology called "lex talionis" initially introduced in v3.2.8, majestic llama can attack back automatically after it successfully preventing one XSS attack

API

mllama.machine()
Get/Set machine
.properties()
Get/Set properties of the machine
.name()
Get/Set machine name
.os()
Get operating system for known machine/Detect and set operating system for new MLMachine object
.ram()
Get RAM/Detect and set RAM
.port()
Get/Set outgoing ports
.mount()
Get mounted devices/Mount device
.unmount()
Unmount device(s)
.encrypt()
Encrypt file object(s) if parameter(s) passed, else machine itself
.decrypt()
Decrypt file object(s) if parameter(s) passed, else machine itself
mllama.environment()
Obtain/Configure running environment for majestic llama
.details()
Get/Set environment properties
.sudo()
Get whether majestic llama is running under super user/Set the super user permission
.GPU()
Get/Set use of GPU computing power, may cause unexpected error
.mode()
Get/Set mode of firewall, include XSLlama, MLlama, LLlama, XLLlama
.time()
Get/Set running time limit
.timing()
Get actual running time
mllama.status()
Return boolean
mllama.firewall()
Get majestic llama firewall
.start()
Start firewall service
.stop()
Stop firewall service
.restart()
Restart firewall service
mllama.autoAnti()
Turn on/off auto attack back
mllama.force()
Force mode, need manual control and super user permission
.firewall()
Force start majestic llama
.attack()
Force attack machine(s) specified
mllama.update()
Do nothing. majestic llama determines when to update itself based on its mood
mllama.unload()
Detach all majestic llama related functionalities
llama.log()
Get log since installation time
.since()
Specify starting time of the log records
.until()
Specify ending time of the log records
.query()
Query on log using SQL
mllama.console()
Console mode. Accept customized commands
mllama.THE_BEAST()
Hidden command. Unknown results
